# DIKWP IntentAsset Management OS 2026 V1

A standalone, offline-first reference platform for operating and governing intention assets in the future intention economy.

## Core idea

An intent asset is a purpose-bound, authorized, revocable and auditable DIKWP object. It is not raw attention, not covert profiling and not a license to manipulate users.

## Workflow

Signal -> Intent elicitation -> Intent Contract -> DIKWP Content Card -> Intent Asset Registry -> Offer Proof Routing -> Action Gate -> Outcome Ledger -> Settlement -> Renewal / Revocation -> IntentAsset Passport.

## Quick start

Open `index.html` in a browser. No server, API key, login, database or network is required.

Optional CLI:

```bash
python tools/run_intent_asset_eval.py examples/sample_intent_asset.json --out outputs
```

## Boundary

The open core does not connect to payment, ad bidding, trading, official portals, identity providers, browser storage or production systems. It does not produce legal, accounting, securities, advertising, credit or medical conclusions.
